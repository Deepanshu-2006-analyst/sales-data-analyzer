# 📊 Sales Data Analyzer

A Python project built using **NumPy** to analyze sales data for multiple products over multiple days. The program calculates revenue statistics, identifies sales trends, and generates a detailed sales report.

## 🚀 Features

- Accept sales data for multiple days and products
- Calculate total revenue
- Calculate average daily revenue
- Identify the highest revenue day
- Identify the lowest revenue day
- Find the best-selling product
- Find the worst-selling product
- Display day-wise revenue
- Display product-wise revenue
- Generate a formatted sales report

## 🛠️ Technologies Used

- Python 3
- NumPy

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/Deepanshu-2006-analyst/sales-data-analyzer.git
```

2. Move into the project folder

```bash
cd sales-data-analyzer
```

3. Install NumPy

```bash
pip install numpy
```

4. Run the project

```bash
python sales_analyzer.py
```
